# Maho Notes Homebrew Tap

Homebrew formulae for [Maho Notes](https://github.com/kuochuanpan/maho-notes) tools.

## Install

```bash
brew tap mahopan/tap
brew install mn
```

## Formulae

| Formula | Description |
|---------|-------------|
| `mn` | Maho Notes CLI — personal knowledge base with multi-vault, FTS5+CJK search, GitHub sync |

## Usage

```bash
mn --help          # standard CLI help
mn skill           # AI-agent-friendly usage guide
mn skill --json    # machine-readable guide
```

## Requirements

- macOS 15.0+
- Swift 6.0+ (installed automatically via Xcode CLT)
